# Introdução

Flutter é uma ferramenta criada pelo Google para desenvolvimento de aplicativos para 6 plataformas diferentes: 

**Android, iOS, Web, Windows, Linux e macOS**

E o melhor de tudo: utilizando o mesmo código-fonte.
